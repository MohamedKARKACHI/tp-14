# Application Android Client SOAP

Application Android consommant un service web SOAP.

## Structure

- `app/` - Application Android (client SOAP)
- `soap-service/` - Service SOAP Spring Boot

## Technologies

- Android SDK
- kSOAP2
- Spring Boot (backend)

## Démarrage

1. Démarrer le service SOAP :
```bash
cd soap-service
mvn spring-boot:run
```

2. Lancer l'application Android dans l'émulateur
